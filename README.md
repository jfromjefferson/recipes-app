# Pacotes utilizados
- Getx  - Navegação, gerência de estado
- hive e hive_flutter - Banco de dados
- flutter_svg - Renderizar svg
- dio - Requisições
- flutter_staggered_grid_view - Organizar os cards na tela com +- 2 items por linha
- admob_flutter - Anúncios
